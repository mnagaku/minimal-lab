
# Minimal Jupyter Lab Stack

minimal-notebook + jupyterlab = minimal-lab

## Basic Use

```
docker run -d --name minimal-lab -p 8888:8888 mnagaku/minimal-lab start-notebook.sh --NotebookApp.token=''
```
